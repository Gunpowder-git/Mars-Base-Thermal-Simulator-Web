# 火星基地隔热墙 · 元胞自动机热传导演示

这是一个基于 **元胞自动机 Cellular Automaton** 的热传导可视化项目。项目以“火星基地隔热墙设计”为情境，通过网页交互仿真展示不同隔热材料、热源位置和结构缺陷对温度分布与保温效果的影响。项目适合用于物理教学、数值仿真实验、科创展示或 AI 辅助编程项目展示。

---

## 项目简介

在火星基地环境中，室内需要保持相对稳定的温度，而外部环境极其寒冷。为了减少热量流失，本项目将基地墙体划分为多个元胞，并让每个元胞根据周围元胞的温度变化进行更新，从而模拟热量在不同材料中的扩散过程。

仿真空间被划分为三个主要区域：

- **0m - 20m：室内区域**
- **20m - 80m：四段隔热墙体**
- **80m - 100m：外太空区域**

用户可以选择不同材料组合，启动仿真后观察温度场如何随时间变化。

---

## 核心原理

项目使用二维元胞自动机模拟热传导过程。

每个元胞的温度会受到上下左右四个相邻元胞的影响，更新公式近似为：

```text
T_new = T + α · dt / dx² · (T_up + T_down + T_left + T_right - 4T)
```

其中：

- `T` 表示当前元胞温度
- `α` 表示材料热扩散系数
- `dt` 表示时间步长
- `dx` 表示空间步长

这个规则可以看作是热传导方程的离散化形式：

```text
∂T / ∂t = α∇²T
```

## 功能特点

### 温度场可视化

项目使用热力图展示二维温度分布：

- 暖色表示温度较高
- 冷色表示温度较低
- 可以直观看到热量如何从室内向墙体和外太空扩散

### 四段墙体材料设计

隔热墙被分成四段，用户可以为每一段选择不同材料，例如：

- 空气
- 混凝土
- 保温泡沫
- 气凝胶
- 陶瓷
- 金属绝缘层
- 真空/太空
- 铜导热层

### 预设方案对比

项目内置多种墙体设计方案：

- 基础保温
- 进阶保温
- 最优设计

用户可以快速切换方案，对比不同材料组合的隔热效果。

### 热源交互

用户可以调节热源温度，也可以拖动热源位置，观察热源靠近墙体或室内不同位置时温度分布的变化。

### 缺陷模拟

项目支持添加墙体缺陷：

- **热桥**：使用高导热材料模拟局部热量快速传递
- **裂缝**：模拟墙体中出现空气缝隙后的温度变化

### 实时物理量显示

界面会实时显示多个物理量：

- 室内均温
- 温度波动
- 墙体均温
- 热流密度
- 墙体温差
- 有效热阻
- 材料成本
- 总重量

### 温度剖面图

下方曲线图展示墙体中心线上的温度变化趋势，帮助分析热量从室内到外太空的传递过程。

## 如何运行

本项目是纯前端项目，不需要安装额外依赖。

下载项目后，直接用浏览器打开：

```text
mars_wall_simulation.html
```

## 教学意义

这个项目将抽象的热传导过程转化为可视化、可交互的动态模型，可以帮助理解：

- 热传导的基本规律
- 材料热扩散系数对温度变化的影响
- 隔热材料组合的工程意义
- 热桥和裂缝等结构缺陷对保温效果的破坏
- 数值仿真在物理建模中的应用
